# How to run the proyect

1. Install all the dependencies using

```console
npm install
```

2. Rename __.env.template__ to __.env__ and fill all the fields

3. Create the mongo database with the command

```console
docker-compose up -d
```

4. Start the proyect using

```console
npm run start 
```
